# istar

A white-and-green themed company website with sections:
- Lights
- Car Accessories
- Sound & Speaker
- TV
- Home Ware
- Kitchen Appliances

Publishing on GitHub Pages:
1) Create a GitHub repository named istar.
2) Add the starter files above (index.html, assets/, js/, etc.).
3) Commit and push to main branch.
4) In GitHub: Settings > Pages > Source: main (root) > Save.
5) Visit https://<your-username>.github.io/istar/

Customization ideas:
- Replace placeholder product cards with real data.
- Add a simple product catalog, contact form, about page.
- Connect to a static JSON feed for dynamic content.
- Add a custom domain via Settings > Pages > Custom domain (and DNS config).
